---

## 🔄 Cập nhật tự động

Đảm bảo script có các trường metadata sau:

```javascript
// @downloadURL https://github.com/hexros-dev/shv-redirect/raw/master/userscript.user.js
// @updateURL   https://github.com/hexros-dev/shv-redirect/raw/master/userscript.meta.js
```

Khi phát hành phiên bản mới, trình quản lý userscript sẽ tự động kiểm tra và cập nhật.

---

## 🛠️ Sử dụng

1. Cài đặt userscript.
2. Mở trang web được hỗ trợ.
3. Script sẽ tự động hoạt động.
